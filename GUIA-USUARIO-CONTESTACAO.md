# 🛡️ Guia do Usuário - Como Contestar uma Operação

## 🎯 O que é Contestação?

A contestação permite que você reporte problemas quando a outra parte não cumpre o acordado em uma operação P2P.

## 🔍 Quando Posso Contestar?

Você pode contestar uma operação quando:
- ✅ Você é participante da operação (criou ou aceitou)
- ✅ A operação está em andamento ou aguardando conclusão
- ✅ A outra parte não cumpriu o combinado
- ✅ Você aceitou os termos de uso do bot

## 📱 Como Contestar (Passo a Passo)

### **Passo 1: Localize o Botão**
Procure pelo botão **"⚠️ Contestar"** na mensagem da operação.

### **Passo 2: Clique no Botão**
Toque no botão **"⚠️ Contestar"** para iniciar o processo.

### **Passo 3: Escolha o Motivo**
Selecione o motivo que melhor descreve o problema:

| Botão | Quando Usar |
|-------|-------------|
| 💰 **Não efetuou pagamento** | A outra parte não enviou o dinheiro |
| 📦 **Não entregou ativos/serviços** | Não recebeu as criptomoedas ou serviços |
| 💸 **Valor incorreto** | Recebeu valor diferente do combinado |
| 🚨 **Tentativa de fraude** | Suspeita de golpe ou má-fé |
| 💬 **Problema de comunicação** | Dificuldades para se comunicar |
| 📋 **Violação dos termos** | Quebra das regras acordadas |

### **Passo 4: Confirmação**
Após selecionar o motivo, você verá uma mensagem de confirmação:

```
✅ Operação Contestada com Sucesso!

Motivo: A outra parte não efetuou o pagamento

A operação foi marcada como em disputa e será 
analisada pelos administradores.

Você receberá uma notificação quando houver 
uma resolução.
```

## ⚠️ Importante Saber

### **Contestações Falsas**
- 🚫 **Não faça contestações falsas** - isso pode resultar em penalidades
- 🚫 **Não conteste por impaciência** - aguarde o prazo combinado
- 🚫 **Não use para cancelar** - use o botão de cancelar se disponível

### **O que Acontece Depois**
1. **📢 Notificações automáticas** - Todas as partes são notificadas imediatamente
2. **⏸️ Operação pausada** - Fica marcada como "em disputa"
3. **🔍 Análise administrativa** - Moderadores irão investigar
4. **⚖️ Resolução** - Decisão será tomada baseada nas evidências
5. **📞 Notificação final** - Você será informado do resultado

### **Notificações que Você Receberá:**
- ✅ **Confirmação imediata** da contestação
- 📧 **A outra parte é notificada** sobre a disputa
- 👥 **Administradores são alertados** automaticamente
- 🔄 **Mensagem do grupo é atualizada** com status de disputa

### **Tempos de Resposta**
- ⏱️ **Análise inicial:** 24-48 horas
- ⏱️ **Resolução completa:** 3-7 dias úteis
- ⏱️ **Casos complexos:** Até 15 dias

## 🚨 Quando NÃO Contestar

### **Situações Normais:**
- ⏳ **Aguardando confirmação** - Dê tempo para a outra parte
- 💬 **Negociando detalhes** - Continue a conversa no privado
- 🔄 **Processamento em andamento** - Transações podem demorar

### **Use Outros Recursos:**
- 🔄 **Reverter operação** - Se ambos concordam em desfazer
- ❌ **Cancelar operação** - Se ainda não foi aceita
- 💬 **Chat privado** - Para resolver mal-entendidos

## 📞 Suporte

### **Precisa de Ajuda?**
- 📖 **Leia os termos:** `/termos`
- 🆘 **Ajuda geral:** `/help`
- 👥 **Suporte:** Entre em contato com administradores

### **Dúvidas Frequentes**

**P: Posso cancelar uma contestação?**  
R: Não, após confirmar o motivo, a contestação será processada. Pense bem antes de contestar.

**P: E se eu contestar por engano?**  
R: Entre em contato com os administradores imediatamente para explicar a situação.

**P: Quantas vezes posso contestar?**  
R: Cada operação pode ser contestada apenas uma vez. Contestações excessivas podem resultar em penalidades.

**P: O que acontece com minha reputação?**  
R: Contestações legítimas não afetam sua reputação. Contestações falsas podem resultar em penalidades.
